Bakery Website Rework

HTML CSS JavaScript

This project transforms Pinecrest Bakery's cluttered website...

This project transforms Pinecrest Bakery's cluttered website into a streamlined online ordering platform, consolidating their fragmented ordering systems into one user-friendly interface. The original website struggled with split functionality between DoorDash and direct ordering, creating a confusing user experience with overwhelming information architecture. To solve this, I developed a unified platform that combines all menus (catering, seasonal, bakery, and cafe) into one intuitive system with an intelligent cart that recognizes different preparation times for various order types. Using modern web technologies and starting with Figma wireframes, I built a responsive design that simplifies navigation and integrates all ordering capabilities while maintaining the bakery's brand identity. The result is a practical, ordering-focused experience that transforms a marketing-centric website into a robust e-commerce platform.

Train Times Today Mobile Application Prototype

FIGMA

As demonstrated in my functional Figma prototype, TTT is a sophisticated...

As demonstrated in my functional Figma prototype, TTT is a sophisticated mobile application I designed to revolutionize the daily commute experience for train passengers through intelligent journey planning and real-time updates. The project showcases my expertise in creating intuitive user interfaces with features like smart location integration, customizable alert systems, and weather-aware route planning - all while maintaining strict security and accessibility standards. I implemented thoughtful micro-interactions and user flow optimizations that reduced commute planning time and increased predictability, demonstrated through features like two-factor authentication, biometric login support, and progressive information disclosure. The app's success lies in its user-centric design approach, where I developed clean layouts, clear visual hierarchies, and streamlined navigation patterns that transformed complex commuting challenges into a smooth, predictable experience for daily train commuters.

Education Portal Replication

HTML CSS JavaScript

The clone begins with a minimalist landing page featuring...

The clone begins with a minimalist landing page featuring a login button and logo, maintaining the University of Miami's visual identity. When clicked, the button directs users to a dedicated authentication page where they enter their credentials through a carefully styled form that matches Blackboard's design language. After successful credential entry, users are redirected to the main Blackboard dashboard interface, which includes a functional navigation system and inner pages. The sequential three-page flow (landing → login → dashboard) accurately mirrors the authentic Blackboard user experience while maintaining consistent styling throughout.
